What can I search for?

The search form will accept any plain-text input and attempt to return relevant episode data. Examples of input can be:

  • Name of a guest
  • Name of a host
  • Name of car featured/reviewed
  • Name of a challenge/special/location
  • Etc.

What can't I search for?

While I've done my best to build a comprehensive database of detailed episode data, some less obvious "metadata" terms may not (yet!) yield matches.

For example, searching for "Antarctica" will return the "North Pole" episode. However, searching for "Hamster" will not return matches for "Richard Hammond".
